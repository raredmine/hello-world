### Regular Expression Precedence
| Regular Expression Feature | Example |
| ------------- | ------------- |
| Parentheses (grouping or capturing)  |  (…), (?:…), (?<LABEL>…) |
| Quantifiers | a*, a+, a?, a{n,m} |
| Anchors and sequence | abc, ^, $, \A, \b, \z, \Z |
| Alternation | a|b|c |
| Atoms | a, [abc], \d, \1, \g{2} |
